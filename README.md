# KITT Franco-Belge

Projet d’intelligence artificielle locale inspirée de la série Knight Rider.

## Objectif
Créer une IA embarquée fonctionnant entièrement en local sur Jetson Nano / Orin / Xavier,
avec interaction vocale, personnalité KITT ou KARR, et accès réseau domestique.

## Fonctions prévues
- Reconnaissance vocale locale
- Synthèse vocale française
- LLM local optimisé Jetson
- Interface web type Jarvis
- Mode hors-ligne complet

## Créateurs
Projet conçu par Manix et son IA.
